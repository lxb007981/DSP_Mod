# StationChargeBoost

Mod for Dyson Sphere Program. BepInEx required.

Increases the maximum charge power for interstellar stations beyond the default limit.

## Usage

Put `StationChargeBoost.dll` into `...\Dyson Sphere Program\BepInEx\plugins`.

## Configuration

The mod adds a configurable option `MaxChargePower` (default: 1000 MW) to control the maximum charge power for interstellar stations.

You can adjust this option after running the game once with the mod installed, in `...\Dyson Sphere Program\BepInEx\config\lxb007981.dsp.StationChargeBoost.cfg`.
